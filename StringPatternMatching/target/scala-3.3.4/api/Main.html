Computes the failure function for an Aho-Corasick automaton.

Computes the failure function for an Aho-Corasick automaton.

The failure function is a key component of the Aho-Corasick algorithm, used to handle mismatched inputs during pattern matching by finding the longest suffix that is also a prefix.

Value parameters

states

A map representing the automaton's states. Each state has a unique ID and contains its successor states (transitions). - states(0) is assumed to be the root state.

def main(): Unit

The entry point for the Finite State Machine (FSM) demonstration.

The entry point for the Finite State Machine (FSM) demonstration.

This method:

  1. Defines a test search string (TEST_SearchString) and a list of keywords (TEST_Keywords).
  2. Initializes a FiniteStateMachine instance with the search string and keywords.
  3. Executes the pattern-matching mechanism (PMM) of the FSM to identify matches.
  4. Prints the results of the FSM's pattern-matching function using prettyprint.
